# 运河文化探索

一个基于 React + Vite 构建的静态网站，展示运河文化相关内容，包含三个主要板块：

## 项目结构

### 板块一：运河千帆/运河舆图
- 基于真实运河走向绘制的地图
- 标注拟开放剧本的城市坐标（徐州、扬州、杭州等）
- 点击地标可进入该城市剧本杀的"航行密档"

### 板块二：航行密档
- 每个剧本杀中涉及的关键运河城市、历史名词、文物、非遗文化等设立独立词条
- 配以图文、音频解说（统一格式）
- "谜案考据"栏目：披露剧本创作背后的历史原型与考据过程

### 板块三：渡客长卷
- 以卷轴方式呈现人物图册（如《循舟纪之南京》包含五个角色）
- 人物关系网构建（类似于人物元宇宙）

## 技术栈

- React 19
- Vite 7
- React Router DOM
- CSS3 (响应式设计)

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## GitHub Pages 部署

项目已配置 GitHub Actions 自动部署到 GitHub Pages。

1. 确保仓库名称为 `Record`（或修改 `vite.config.js` 中的 `base` 路径）
2. 在 GitHub 仓库设置中启用 GitHub Pages
3. 选择 `gh-pages` 分支作为源
4. 推送代码到 `main` 分支后，GitHub Actions 会自动构建并部署

## 注意事项

- 图片部分使用图形+文本占位，可根据需要替换为真实图片
- 音频部分为占位内容，需要添加实际的音频文件
- 数据为模拟数据，可根据实际需求进行扩展
